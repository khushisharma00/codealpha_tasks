# Basic Network Sniffer

## Description

This project is a Basic Network Sniffer developed using Python and Scapy. It captures live network packets and displays important information about each packet.

## Features

- Captures live network traffic
- Displays source IP address
- Displays destination IP address
- Detects TCP and UDP protocols
- Displays source and destination ports
- Shows packet length
- Displays the current timestamp

## Technologies Used

- Python 3
- Scapy

## Installation

1. Clone the repository.
2. Install dependencies:

```bash
pip install -r requirements.txt
```

## Run the Project

```bash
sudo python network_sniffer.py
```

## Sample Output

- Source IP
- Destination IP
- Protocol
- Source Port
- Destination Port
- Packet Length
