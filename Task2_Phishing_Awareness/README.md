# Phishing Awareness Training

## Description

This project is a cybersecurity awareness presentation that explains phishing attacks, common phishing techniques, warning signs, prevention methods, and safety tips.

## Topics Covered

- Introduction to Phishing
- Types of Phishing
- Real-life Examples
- Fake Websites
- Prevention Tips
- Interactive Quiz

## Tools Used

- Microsoft PowerPoint / Google Slides
- PDF
